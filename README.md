## Installation
```
git clone https://github.com/basakstuff/twitter-clone.git
cd twitter-clone-app
npm install
npm start
```

![](https://github.com/basakstuff/twitter-clone/blob/master/Project.gif)

## Tech
- ReactJS
- Tailwindcss
- Firebase Firestore
